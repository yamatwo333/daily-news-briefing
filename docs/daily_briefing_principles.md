# Daily Briefing Principles

The foundational document is the map:

- `docs/news_foundation_2026_ja.pdf`
- `docs/news_foundation_2026_ja.docx`
- `docs/news_foundation_2026_ja.md`

The daily brief is the daily guided exercise built on top of that map. It should help the reader place today’s events into the larger structure, without re-teaching the whole foundation from zero.

Use these five reading questions every day:

1. これは何の話か
2. なぜニュースになるのか
3. 誰が得して、誰が困るのか
4. 日本や自分の生活とどう関係するのか
5. 一時的な話か、構造的な話か

Pattern A is the default story mix:

- international politics / geopolitics / security
- economy / prices / interest rates / macro
- Japan domestic politics / policy / institution / social system
- business / industry / technology

Use 4 stories by default. Use 3 on genuinely light days. Use 5 only when one extra story is clearly too important to omit.

Keep the writing beginner-friendly, calm, plain, and non-dense. Explain key terms immediately, prefer short paragraphs, and avoid compressed analyst language.
